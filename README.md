All files are directly downloaded from EQ2U. The files remain unchanged and can be directly imported into your game.


**__STEPS FOR IMPORTING AAs__**
1.) Aquire an AA build you want to import. Feel free to check this form or check EQ2U.com to find one.
2.) Open the file location for your everquest 2 folder
3.) Move the AA build into the everquest 2 directory
4.) Use the ingame "Load AA Build" to import the build.
5.) Recommit the AAs
6.) Dont forget to recast you buffs! 

For any questions feel free to reach out!
